# Ad Banner Images

Please place your restaurant and dish images in this folder with the following names:

## Required Images:

1. **nls-restaurant.jpg** - Restaurant interior photo (Shop profile - First picture)
2. **nasi-lemak.jpg** - Nasi Lemak dish photo (Dish 1)
3. **rendang.jpg** - Rendang dish photo (Dish 2)
4. **laksa.jpg** - Laksa dish photo (Dish 3)
5. **laksa-2.jpg** - Laksa Special dish photo (Dish 4)
6. **lontong.jpg** - Lontong Sayur dish photo (Dish 5)

## Image Requirements:

- **Format**: JPG, JPEG, or PNG
- **Recommended size**: 400x400px minimum (square format works best)
- **File size**: Keep under 500KB per image for fast loading
- **Aspect ratio**: 1:1 (square) is recommended

## How to Add Images:

1. Copy your images to this folder: `frontend/public/images/ads/`
2. Rename them to match the file names listed above
3. The ad banner will automatically display them once the files are in place

Note: If images are missing, the banner will show colorful gradient placeholders with emoji icons.
