RAILWAY QUICK DEPLOY - Step by Step (5 minutes)
═══════════════════════════════════════════════════════

STEP 1: Go to Railway
───────────────────────────────────────────────────────
https://railway.app

Sign up with GitHub (free, takes 30 seconds)


STEP 2: Deploy from GitHub
───────────────────────────────────────────────────────
1. Click "New Project"
2. Select "Deploy from GitHub repo"
3. Choose: zkswol/WheelEat
4. Click "Deploy Now"


STEP 3: Configure Service
───────────────────────────────────────────────────────
1. Click on the service (should be auto-created)
2. Click "Settings" tab
3. Find "Root Directory"
4. Click "Override"
5. Type: backend
6. Click "Save"


STEP 4: Set Start Command
───────────────────────────────────────────────────────
1. Still in Settings
2. Find "Start Command"
3. Type: uvicorn main:app --host 0.0.0.0 --port $PORT
4. Click "Save"


STEP 5: Add Environment Variable
───────────────────────────────────────────────────────
1. Go to "Variables" tab
2. Click "New Variable"
3. Name: FRONTEND_URL
4. Value: https://wheeleat.vercel.app
5. Click "Add"


STEP 6: Get Your Backend URL
───────────────────────────────────────────────────────
1. Go to "Settings" tab
2. Scroll to "Domains"
3. Railway gives you a URL like: https://xxx.railway.app
4. COPY THIS URL!


STEP 7: Update Frontend (Vercel)
───────────────────────────────────────────────────────
1. Go to: https://vercel.com/ybtan6666s-projects/wheeleat/settings
2. Click "Environment Variables"
3. Click "Add New"
4. Name: REACT_APP_API_URL
5. Value: https://xxx.railway.app (paste your Railway URL)
6. Environment: Check all (Production, Preview, Development)
7. Click "Save"


STEP 8: Redeploy Frontend
───────────────────────────────────────────────────────
1. Go to "Deployments" tab in Vercel
2. Click "⋯" (three dots) on latest deployment
3. Click "Redeploy"
4. Wait 2-3 minutes


STEP 9: Test!
───────────────────────────────────────────────────────
1. Test backend: https://xxx.railway.app/
   Should see: {"message":"WheelEat API is running","status":"ok"}

2. Test frontend: https://wheeleat.vercel.app
   Should load data from backend!

═══════════════════════════════════════════════════════

DONE! Your backend should now work!
