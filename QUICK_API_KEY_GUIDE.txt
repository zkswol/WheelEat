========================================
QUICK GUIDE: Get Google API Key
========================================

STEP 1: Go to Google Cloud Console
   → https://console.cloud.google.com/
   → Sign in with your Google account

STEP 2: Create or Select Project
   → Click "Select a project" dropdown
   → Click "NEW PROJECT"
   → Name it: WheelEat
   → Click "CREATE"

STEP 3: Enable APIs
   → Go to "APIs & Services" → "Library"
   → Search "Places API" → Click "ENABLE"
   → Search "Maps JavaScript API" → Click "ENABLE"

STEP 4: Create API Key
   → Go to "APIs & Services" → "Credentials"
   → Click "+ CREATE CREDENTIALS" → "API key"
   → COPY the key (looks like: AIzaSy...)

STEP 5: Add to Project
   → Open: backend-node\.env
   → Replace: YOUR_API_KEY_HERE
   → With: your actual API key
   → Save file

STEP 6: Restart Backend
   → In backend window: Ctrl+C
   → Then: npm start

STEP 7: Test
   → Open: http://localhost:8001/restaurants
   → Should see restaurant data!
